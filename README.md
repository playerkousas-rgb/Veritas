# 📸 Veritas (真境) 
### The Anti-AI Camera Project

**Veritas** 是一個致力於恢復照片「真實性」的相機 App。在這個 AI 氾濫的時代，我們只記錄真實。

## 🛠️ 技術特點
- **硬體驗證**：自動記錄拍照時的物理傳感器數據。
- **紙質感 UI**：模擬傳統攝影集的極簡視覺設計。
- **反 AI 標誌**：每張照片都附帶不可篡改的真實性認證
